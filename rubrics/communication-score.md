# Communication Quality Score

Each category is scored from 0 to 10.

- Clarity: can a real buyer understand it quickly?
- Specificity: does it avoid vague claims?
- Credibility: does it sound believable?
- Differentiation: does it avoid category cliches?
- Customer Language Fit: does it reflect how the market actually speaks?
- Offer Strength: is the value proposition concrete?
- Anti-Hype: does it avoid inflated language?
- Response Potential: does it make a next step feel natural?

Final CQS is the sum of all category scores.
