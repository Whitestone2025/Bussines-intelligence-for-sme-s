# Codex UI Editorial Briefing V2

## Vision

Replace the current Codex Business OS interface with a true executive briefing experience in Spanish. The product must feel closer to a strategy report and working room than to a dashboard or repo browser.

## Outcome

- One active case at a time
- Executive storytelling first
- Audit path second
- Spanish-first visible experience
- The Preview used as flagship briefing case

## Experience Model

### Primary reading

- Portada ejecutiva
- Tesis
- Cliente
- Mercado
- Competencia
- Viabilidad
- Decision
- Plan
- Documentos

### Secondary reading

- Auditoria

## Quality Bar

- The UI must not resemble the current sidebar-driven workspace.
- Executive mode must avoid raw ids, file language, and uncurated artifact labels.
- The Preview should be understandable in under 10 minutes.
- Document reading should feel premium and calm.
- Desktop and mobile must both work.

## Validation

- compileall
- case view-model contract
- UI workspace flow
- business OS end-to-end
- browser QA for portada, cliente, decision, documentos, auditoria
